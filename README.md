### Installation

1. Clone the repository.
    ```bash
    git clone https://github.com/AvazovOgabek/Django-News-Hub.git
    ```
2. Install dependencies.
    ```bash
    pip install -r requirements.txt
    ```
3. Setup the database.
    ```bash
    python manage.py migrate
    ```
